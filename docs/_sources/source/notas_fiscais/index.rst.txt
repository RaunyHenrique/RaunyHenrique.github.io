Controle de notas fiscais
=========================

    No controle de notas fiscais, você podera ter o controle de todas as notas fiscais da sua coleção de jogos.


.. image:: 016.png

Adicionando notas fiscais
^^^^^^^^^^^^^^^^^^^^^^^^^^

* Todas as notas ficais são geradas automaticamente ao se inserir um novo jogo no sistema.


Editando notas fiscais
^^^^^^^^^^^^^^^^^^^^^^^

1. Para editar alguma informação de uma nota fiscal clique em:

.. image:: /_static/img/005.png

2. Em seguida atualize o valor do campo que deseja alterar no formulario:

.. image:: 017.png

3. E finalmente clique no botão a seguir para salvar:

.. image:: /_static/img/007.png


Deletando notas fiscais
^^^^^^^^^^^^^^^^^^^^^^^^

* Você não pode deletar uma nota fiscal.
